# Code Book

This Code Book contains the describtion of all the variables present in the final tidy.txt data file.
There are three different variables called:
###1. Activity Labels
###2. Identifiers
###3. Measurements
#

## 1. Activity Labels
* `1` - the person was `WALKING`
* `2` - the person was `WALKING_UPSTAIRS`
* `3` - the person was `WALKING_DOWNSTAIRS`
* `4` - the person was `SITTING`
* `5` - the person was `STANDING`
* `6` - the person was `LAYING`

##2. Identifiers
* `subject` - The person's ID
* `activity` - The type of activity when the measurement was taken
